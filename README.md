# GPIO Driver Outputs



https://github.com/user-attachments/assets/d2a32075-0e66-4f6c-a475-8fb7b8f11999



This video shows the LED blinking on pin 13 using the implemented GPIO driver. The pin is configured as output and toggled using direct register access. The delay between toggles is generated using Timer1 in CTC mode, so the blinking is consistent without using any built-in delay functions.
